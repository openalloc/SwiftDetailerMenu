# SwiftDetailerMenu

Menu support for _SwiftDetailer_, a multi-platform SwiftUI component for editing fielded data.

Available as an open source library to be incorporated in SwiftUI apps.

_SwiftDetailerMenu_ is part of the [OpenAlloc](https://github.com/openalloc) family of open source Swift software tools.

## Features

This is an _optional_ package that can be used to create menus to support _SwiftDetailer_.
